========================================
Welcome to AEPS Platform's documentation
========================================
:dfn:`Agricultura Especifica por Sitio` (AEPS) Platform is a suite of open source tools 
that help organizations collect, store, analyze and visualize information 
about productive events of crops as support in the decision-making, 
under focus of site-specific agriculture.
